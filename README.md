# Khaos Icon Maker

## Install

1. Go to the [releases](https://github.com/QuingKhaos/KhaosIconMaker/releases) page and
   download the source code archive for the latest release.
2. Extract that into your `<your SML starter project>/Plugins/` directory.
3. Open the `IconMaker` level from KhaosIconMaker Content and save as new level into your mod.
